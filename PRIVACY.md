# Privacy Policy

即阅 is a local Android document preview tool.

## Data Collection

即阅 does not collect personal information.

## Network

即阅 does not request the Android `INTERNET` permission. The app is designed to work locally.

When previewing local HTML, 即阅 blocks `http` and `https` network requests from the embedded WebView.

## Local Files

Files opened with 即阅 are copied into the app's private storage so the recent list and favorites can work locally.

For local HTML tools, the WebView may read files that the user explicitly selects through the Android file picker. Those files are not uploaded by 即阅.

Generated files from local HTML tools may be saved to the device Downloads folder when the user triggers a download.

## Accounts

即阅 does not provide accounts, login, sync, analytics, ads, or payment features.
